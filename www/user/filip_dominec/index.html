Symbols

Common SMD components

These symbols are useful if you use surface mount transistors in the standard SOT23 package, which have often the "1-B, 2-E, 3-C" pins assignment. Similar holds for small unipolar transistors. (For example, I use often: BC850, BCW66H, BCW68H, BC859, 2N7002.)

Before I made these symbols, I had to manually edit the netlist file so that the "Q1-B" pin is renamed to "Q1-1" etc. Otherwise the autorouter did not route to the pin.

Similar issues still occur e. g. for polarized capacitors on some non-capacitor-specific footprints. To fix an existing file, you may use global find and replace. On Linux, save following command to a script and run it, passing the netlist filename as a parameter: cp $1 $1.old; cat $1.old | sed -e 's/\(C[0-9]\+\)-N/\1-1/g' -e 's/\(C[0-9]\+\)-P/\1-2/g' > $1.

  • NPN_SOT23_BEC.sym - a common bipolar NPN transistor
  • PNP_SOT23_BEC.sym - a common bipolar PNP transistor
  • NMOS_SOT23_GSD.sym - a common unipolar N-channel transistor
  • PMOS_SOT23_GSD.sym - a common unipolar P-channel transistor
