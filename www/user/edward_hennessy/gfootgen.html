Footprint Generator

This footprint generator allows the user to quickly generate footprints for use with PCB.

Installation

The project source is stored in GitHub here:

https://github.com/ehennes775/gfootgen

Here are the packages that should be installed.

  • ruby
  • ruby-atk
  • ruby-cairo
  • ruby-gdk3
  • ruby-gdk-pixbuf2
  • ruby-gio2
  • ruby-glib2
  • ruby-gtk3
  • ruby-pango

Currently, the footprint generator is distributed as a Ruby source. Obtain the source from GitHub.

$ git clone https://github.com/ehennes775/gfootgen.git

Enter the top level directory and build the gem.

$ cd gfootgen
$ gem build gfootgen.gemspec

After building the gem, install the gem.

$ sudo gem install gfootgen-0.0.1.gem

The program can be launched at the command line:

$ gfootgen

Usage

Select Change button to select the type of footprint to construct.

Adjust the footprint parameters at the left.

Numeric parameters accept both mm and mil for units. Any parsing errors will default to mm.

Package Length The dimension of the package on the x axis. Used to calculate the silkscreen border.
Package Width The dimension of the package on the y axis. Used to calculate the silkscreen border.
Pad Length The dimension of the copper pad on the x axis.
Pad Width The dimension of the copper pad on the y axis.
Pad Spacing (BSC) The dimension between pads on opposite sides of the device, from center to center. Negative values will flip the device horizontally.
Pad Pitch (BSC) The dimension between pads in the same row, from center to center. Negative values will flip the device vertically.
Pin Count The total number of pins/pads on the device.
Pad Clearance The minimum distance from the copper pad to surrounding copper.
Pad Soldermask Relief The minimum distance from the copper pad to the soldermask opening.
Silkscreen Package Offset The additional distance from the edge of the package to the centerline of the silkscreen.
Silkscreen Pad Offset The additional distance from the edge of the pad copper to the centerline of the silkscreen.
Silkscreen Thickness The silkscreen line thickness.
Silkscreen Outline Selects the type of outline for the footprint (e.g. polarized, non-polarized).

The footprint generator provides both a graphic and text preview of the footprint, allowing the user a preliminary check for correctness.

When complete, click the Export button and save the footprint to a file.
