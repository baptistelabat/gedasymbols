These symbols are for the Analog Devices Blackfin DSP. They were originally created for use in DSPcard. This symbol is broken into six files. They should all be given the same reference designator, and only one needs a footprint to be specified.

ADSP-BF532 (LQFP-176 package) also ADSP-BF531

  • Memory bus
  • PPI, SPI, GPIO
  • SPORTs
  • Clock, reset, misc
  • Power supplies
