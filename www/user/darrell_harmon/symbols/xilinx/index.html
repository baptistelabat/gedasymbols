These symbols are for Xilinx FPGAs. They were originally created for use in DSPcard. These symbols are broken into many files (10 for Spartan 3 FT256). They should all be given the same reference designator, and only one needs a footprint to be specified.
These symbols were created using CSV files provided by Xilinx, so should be accurate.
