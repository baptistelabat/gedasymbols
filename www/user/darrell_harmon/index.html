All of these shematic symbols are designed to be as compact as possible. They may not conform to gschem standards.

Tools

  • Footgen - A Python script which generates footprints based on a text description. I am currently devloping a web based version which will be available soon.
  • mksym.c - Generates a schematic symbol from a csv file listing the pin names and numbers. Pins are placed in order in the file. It should be re arranged using gschem to produce a usable symbol.
