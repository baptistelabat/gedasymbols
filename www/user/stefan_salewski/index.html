All my symbols and footprints are licensed GPL for distribution, and unlimited for use. They are created with care, but most are still untested on real hardware. Larger rectangular symbols are created with the Python script Tragesym. Some symbols uses footprints of John Luciani's footprint page or my own footprints (see below). Most of my footprints are generated with the Python script Footgen. (For some footprints an enlarged version with name extension _XL optimized for hand soldering exist.)
