Tools

  • nice_sym.py: a script that helps to beautify and repair gschem symbols. It creates the same symbol style as tragesym
    • set attribute colors
    • add templates of missing attributs
    • set pin attribute allignments and colors
    • add templates of missing pin attributes
  • subdir_gsymcheck.sh: a shell script that runs gsymcheck in all subdirectories that contain one or more gEDA symbols.
