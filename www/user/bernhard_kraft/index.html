Symbols

  • Titleblock A4 vertical/portrait (title-A4-vert.sym)
    This titleblock features a A4 sheet in portrait format. The labels are at the bottom of the sheet and are implemented using attributes. So you can easily replace the default values in one place by using the "Edit attributes" dialog (Right click "Edit..." onto a symbol).

  • Generic LDO (low dropout) regulator (LDO.sym)
    There are already quite a bunch of 78xx regulator icons. This 78xx.sym for example is mostly the same as this LDO.sym regulator. Usually nowadays drop-in DC-DC regulators allow a 78xx to get replaced by a more efficient, power saving DC-DC regulator. At least for 78xx devices which use an upright TO220 footprint/case.

  • Microchip MCP79400 Real-time-clock (MCP79400.sym)
    Symbol for Microchip RTC MCP79400 with serial 2-wire interface, I2C(TM) compatible.

  • Microchip PIC 18F25K80 (pic18F25K80.sym)
    Symbol for Microchip PIC microcontroller 18F25K80 (specialities: CAN bus interface) datasheet

  • SD card, raw symbol (sd-card-1.sym)
    Only the 9 connectors of the SD card itself.

  • SD card connector with insertion/write-protect switch (sd-card-2.sym)
    Typical connectors like the Hirose SD card connectors also feature pins for insertion/removal detection and for determining the state of the write protect switch.

  • Sensirion SHT21 humidity/temperature sensor (SHT21.sym)
    New generation humidity/temperature sensor from Sensirion with digital I2C(TM) interface. datasheet
