Tools

Documentation

Symbols

  • C64/C128 Expansion Port - Derived from stock GEDA 40-pin header/connector.
  • SJI3533NG - 3.5mm Stereo Jack, companion to the similarly-named footprint below (CUI, Inc.).
  • TC7WU04F - Three invertors in one package (Toshiba)
  • TLC-7226CN - Quad 8-bit Digital to Analog converter (Texas Instruments)

Footprints

Common parts:

  • Miniature capacitor - 100 mil lead spacing, not sure if any such parts actually exist.
  • Basic ceramic capacitor - 200-mil lead spacing, typically decoupling capacitors.
  • Basic resistor - 1/8 watt (maybe 1/4 also), vertical, 100-mil lead spacing. Should work for small signal diodes too.
  • Basic Transistor - 2N2222 or similar, standard TO-18 case, derived from Eagle's part.
  • Common Mounting hole - A simple screw hole, 350 mil pad diameter, 130 mill drill (as a component so PCB won't move it around).

IC's:

  • Basic 4016/1066 Switch - Derived from standard DIL14 generic footprint.
  • 74LS74 - Common 74LS74 Flipflop, 14-DIL package.
  • 744040 - 12-bit Counter IC with Reset, Derived from generic DIL16 footprint.
  • MXD1000 - 4 to 500 ns Delay line, derived from generic DIL8 footprint (Maxim).
  • CY7C138 - Dual-Port 4Kx8 SRAM by Cypress, derived from generic PLCC68 Socket footprint (Cypress).
  • MOS 6581 SID chip - derived from generic DIL28 footprint, with proper pin names.

Connectors:

  • 2-794954-0 - Female ATX Connector, as used on most mainboards. 20-pin version, from datasheet (Tyco/AMP).
  • 2-1586041-0 - Female ATX, Right angle, 20 pin version, from datasheet (Tyco/AMP).
  • DC0202 - Female DC Power jack. Generic footprint, should work for most wall warts, derived from Eagle part.
  • Female 5-pin DIN - PCB mounted, suitable for older keyboards, etc. derived from Eagle part.
  • Female 15-pin D-Sub (short) - Shortened version suitable for use at the edge of a board, similar to PCB's other SubD connectors.
  • Female 15-pin D-Sub - Standard HD-15 SubD female VGA connector, converted from Eagle.
  • Female 24-position Edge - Suitable for Commodore 64/128 "User Port" connector.
  • PG203J - Basic 3.5mm stereo jack, derived from Eagle part (Radio Shack).
  • SJ13533NG - Lower-profile 3.5mm stereo jack, without internal switches, based on datasheet (CUI, Inc).
  • Male 44-position Edge - Suitable for Commodore 64/128/etc usage.
