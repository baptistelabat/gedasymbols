Symbols

Common Parts:

  • Generic crystal - Basic 4-pin crystal "XTAL" type oscillator

IC's:

  • TC7WU04F - Three invertors in one package (Toshiba)
  • TLC-7226CN - Quad 8-bit Digital to Analog converter (Texas Instruments)
  • CY7C138 - 4k x 8 dual-port SRAM (Cypress)
  • DS1135 - triple 10ns delay line (Maxim)
  • 74x4040 - 12-bit binary counter with reset
  • LM324 - Quad Op Amp

Connectors:

  • C64/C128 Expansion Port - Derived from stock GEDA 40-pin header/connector.
  • C64/C128 User Port - Derived from Leon Kos' generic 24-pin connector.
  • Stereo Jack - Common two-channel "stereo" connector.
  • Female DIN-5 - Female 5-pin round DIN connector (common, for old AT keyboards, C64 video, etc.
  • PowerJack-2.sym - Common DC power jack.
  • PowerJack-3.sym - Common DC power jack - without dotted lines.

Footprints

Common parts:

  • Basic ceramic capacitor - 200-mil lead spacing, typically decoupling capacitors.
  • Resistor/Diode - 100-mil lead spacing for 1/8 and 1/4W resistors and small signal diodes.
  • Basic Transistor - 2N2222 or similar, standard TO-18 case, derived from Eagle part.
  • Common Mounting hole - A simple screw hole, 350 mil pad diameter, 130 mill drill (as a component so PCB won't move it around).
  • PT10LH01 - Common potentiometer - standing upright, horizontal shaft (Piher)

Connectors:

  • 2-794954-0 - Female ATX Connector, as used on most mainboards. 20-pin version, from datasheet (Tyco/AMP).
  • 2-1586041-0 - Female ATX, Right angle, 20 pin version, from datasheet (Tyco/AMP).
  • Female 5-pin DIN - PCB mounted, suitable for older keyboards, etc. derived from Eagle part.
  • Female 15-pin D-Sub - Standard HD-15 SubD female VGA connector, derived from Eagle part.
  • Female 15-pin D-Sub (short) - Shortened version suitable for use at the edge of a board, similar to PCB's other SubD connectors.
  • Female 24-position Edge - Suitable for Commodore 64/128 "User Port" connector.
  • Female 24-position Edge - Same as above, with a silk line as a connector body width reference.
  • PG203J - Basic 3.5mm stereo jack (the kind Radio Shack sells), derived from Eagle part.
  • SJ13533NG - Lower-profile 3.5mm stereo jack, without internal switches, based on datasheet (CUI, Inc).
  • Male 44-position Edge - Suitable for Commodore 64/128/etc usage.
  • 163-7620-E - 5.5mm outer contact, 2.1mm pin diameter (Kobiconn).
