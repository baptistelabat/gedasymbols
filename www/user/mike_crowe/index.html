Here are some preliminary .xslt files and a bash script that converts an SVG to a PCB element list. I use xsltproc as my tranforming engine, but any xslt tool should work. The bash script takes an input file and pipes it through the two xsl stylesheets. The first sheet attempts to put the svg into a form containing jus the content type <lines>. The second xslt converts the <lines> to a pcb element list.
