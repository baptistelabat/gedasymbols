Symbols

  • A3PN250 FPGA in VQ100 package (Microsemi SOC, formerly Actel):
    • A3PN250 Core
    • A3PN250 I/O bank 0
    • A3PN250 I/O bank 1
    • A3PN250 I/O bank 2
    • A3PN250 I/O bank 3
    • JTAG connector for Microsemi SOC/Actel FPGAs

    Note for the A3PN250 FPGA: You will need the core and all four I/O bank symbols. Give them all the same reference designator (e.g. U3), so gEDA will recognize the five symbols as being from a single FPGA.
