Title blocks

To place more smaller elements on page, title blocks can be scaled by using scale-schematic Perl script:
perl scale-schematic 2 title.sym

  • title-gost-A4-first.sym — first A4 page with GOST title block
  • title-gost-A4-others.sym — others A4 pages with GOST title block
  • title-gost-A3-first.sym — first A3 page with GOST title block
  • title-gost-A3-others.sym — others A3 pages with GOST title block
