PCB Footprint Generators:
These generators are easist to use by checking out the cvs repository. The footprints generated are included in the footprints section below so you should only need to use these generators if you want to modify the footprints they generate. If you want to download the scripts to use, make sure to download the two helper files at the bottom of the list.
  • KK-100.py - Generator for .1" KK Right Angled Connectors
  • KK-156.py - Generator for .156" KK Right Angled Connectors
  • MicroFit_SMT_Press_1Row.py - Generator for 3mm Single Row Microfit Surface Mount Press Fit Right Angled Connectors
  • MicroFit_SMT_Press.py - Generator for 3mm Double Row Microfit Surface Mount Press Fit Right Angled Connectors
  • Telit_GM862.py - Generator for Telit GM862 Cell Modem
  • SSOP_DB.py - Generator for SSOP_DB surface mount ICs
  • AudioStix.py - Generator for Audiostix (or GPSstix) header footprint
  • MicroFit_Through.py - Generator for 3mm Microfit Through Hole Right Angled Connectors
  • ArrayGenerator.py - Helper File for used by some of the generators
  • WriteComponents.py - Helper File for used by some of the generators
